# PortuPrep — Инструкции для Cursor

> Личное приложение для изучения португальского языка (бразильский вариант).
> Стек: **один HTML файл** + IndexedDB для хранения данных в браузере.
> Никакого бэкенда на старте. Внешние API подключаются по мере необходимости.

### Сводка: документ ↔ текущий код (`tutor-app`, `portuprep-extracted.html`)

| Область | В документе | В коде сейчас |
|--------|-------------|----------------|
| Оболочка | Один HTML | Vite + iframe на тот же HTML |
| Словарь | IndexedDB, сортировки, источник | Слова в **памяти страницы** (после перезагрузки сбрасываются); поиск/теги/редакт есть; **нет** сортировок, полей источника видео как в схеме |
| Словарь — доп. | Импорт текста, связанные формы, Unsplash, drag&drop | **Нет** |
| YouTube | Полный sentence mining | Видео + субтитры + синхрон + глоссы; **есть** подсветка слов по словарю (оранж./зел.), пауза по клику на слово, **Video card** из выделения + конструкции по шаблонам; **нет** IndexedDB для карточек, заголовка ролика по API, интеграции карточек в SM-2 |
| Карточки | SM-2, типы карт, режимы ввода/аудио | SM-2 + обычная карточка + TTS; **нет** Video/конструкция/спряжение, режимов ввода/диктанта |
| Тесты | 5 режимов, дистракторы по сходству, пример после ответа | **2 режима** (MC + ввод); дистракторы случайные; пример после ответа **нет** |
| Грамматика | 4 блока темы, SVG-шкалы, Unsplash в темах | Статические темы + **Conjugador**; **нет** шкал, мнемо, «+ в карточки» из спряжения |
| Статистика | CEFR, heat map, графики, слабые места | Простой экран прогресса; **нет** heat map / CEFR / графиков |
| Произношение / PDF / синк | Этапы 2–3 | **Не делалось** |

Идеи из файла **намеренно по этапам** (см. «Порядок реализации»); часть помечена как этап 2–3. Реализовано не «всё сразу», а подмножество этапа 1 + dev-API для субтитров/перевода.

---

## Архитектура

- **Один файл** `portuprep.html` — HTML + CSS + JS всё внутри
- **IndexedDB** — хранение слов, карточек, истории тестов, статистики между сессиями
- **Web Speech API** — голосовой ввод и TTS произношение (бесплатно, в браузере)
- **ElevenLabs API** — качественный бразильский TTS и проверка произношения (платный, подключить позже)
- **ffmpeg.wasm** — экспорт аудио в MP3 прямо в браузере (CDN)
- **YouTube iframe API** — встраивание видео с управлением таймкодом

## Модули приложения

### 1. СЛОВАРЬ

**Основной функционал:**
- Карточка слова содержит: слово (PT), перевод (RU), пример предложения, тег/категория, изображение, источник (из какого видео/текста добавлено)
- Поиск в реальном времени по слову и переводу
- Фильтрация по тегам: глаголы / существительные / прилагательные / конструкции / всё
- Редактирование и удаление слов
- Сортировка: по дате добавления / по алфавиту / по сложности

**Голосовой ввод слова:**
```
Нажал [🎤] → говоришь "falar — говорить"
→ Web Speech API распознаёт
→ парсится: слово = "falar", перевод = "говорить"
→ появляется в форме на проверку
→ [✅ Сохранить] одним тапом
```
Реализация: браузерный `SpeechRecognition`, никакого API не нужно.

**Импорт текста (clipboard / вставка):**
- Вставляешь любой текст (WhatsApp, статья, субтитры)
- Приложение выделяет слова которых **нет в словаре** (оранжевые)
- Показывает частотность каждого слова
- Одним кликом добавляешь нужные

**Связанные слова:**
- К слову `falar` автоматически линкуются `fala`, `falando`, `falei`
- Реализация: простой паттерн-матчинг по корню

**Изображения к словам:**
- Поле URL картинки при добавлении (вставить любую ссылку)
- Кнопка "Найти картинку" → запрос к Unsplash API: `https://api.unsplash.com/search/photos?query={word}`
- Drag & drop своей картинки
- Fallback: `https://source.unsplash.com/featured/?{word_in_english}`

### 2. YOUTUBE + СУБТИТРЫ (Sentence Mining)

**Основной флоу:**
1. Вставляешь YouTube ссылку → видео загружается через YouTube iframe API
2. Рядом — транскрипт с таймкодами (субтитры)
3. Незнакомые слова (которых нет в словаре) **автоматически подсвечиваются** оранжевым
4. Клик на слово → всплывающий попап с переводом + кнопка "В словарь"
5. Видео автоматически паузируется при клике

**Получение субтитров:**
- Начальный вариант (без бэкенда): демо-транскрипт или ручная вставка субтитров
- Продвинутый вариант: Python бэкенд с `youtube-transcript-api`, возвращает JSON с таймкодами

**Создание Video Card (Sentence Mining):**
- Выделяешь мышью кусок текста в транскрипте
- Появляется кнопка **"📌 Создать карточку"**
- Открывается модалка с автоматическим разбором:
  - каждое слово кликабельно
  - зелёные = уже в словаре, оранжевые = новые
  - определяется грамматическая конструкция (паттерн-матчинг)
  - таймкод → ссылка на момент в видео

**Структура Video Card (сохраняется в IndexedDB):**
```json
{
  "type": "video_sentence",
  "videoId": "abc123",
  "videoTitle": "Português com Letícia - EP12",
  "timestamp": 42,
  "sentence": "Eu preciso de mais prática",
  "translation": "Мне нужно больше практики",
  "words": ["preciso", "prática", "mais"],
  "construction": "precisar de + [substantivo]",
  "addedAt": "2026-04-10"
}
```

**Коллекция грамматических конструкций для авто-определения:**
```json
[
  { "pattern": "precisar de", "label": "нуждаться в / нужно", "example": "Preciso de ajuda." },
  { "pattern": "gostar de", "label": "нравится", "example": "Gosto de música." },
  { "pattern": "ir + inf", "label": "будущее время (vou falar)", "example": "Vou estudar." },
  { "pattern": "estar + ndo", "label": "настоящее продолженное", "example": "Estou falando." },
  { "pattern": "acabar de + inf", "label": "только что сделал", "example": "Acabei de chegar." },
  { "pattern": "ter que + inf", "label": "должен", "example": "Tenho que ir." },
  { "pattern": "pode + inf", "label": "можно / могу", "example": "Pode falar." },
  { "pattern": "ficar + ndo", "label": "продолжать делать", "example": "Fico pensando." },
  { "pattern": "deixar de + inf", "label": "перестать делать", "example": "Deixei de fumar." }
]
```

### 3. КАРТОЧКИ (Spaced Repetition)

**Алгоритм SM-2** — как в Anki:
- Три кнопки оценки: ✗ сложно / − нормально / ✓ легко
- Каждая регулирует следующий интервал повторения
- Хранится в IndexedDB: дата следующего повторения, интервал, ease factor

**Типы карточек:**

*Обычная карточка:*
- Лицевая: слово на PT + изображение
- Обратная: перевод + пример предложения
- 3D flip анимация

*Video Card:*
- Лицевая: предложение с пропущенным словом
- Обратная: ответ + кнопка "Переслушать момент из видео" (ссылка с таймкодом)
- TTS произносит предложение

*Карточка конструкции:*
- Лицевая: ситуация ("как сказать 'мне нужно + существительное'?")
- Обратная: конструкция + пример + изображение

*Карточка спряжения:*
- Лицевая: глагол + лицо + время ("FAZER — eu, pretérito")
- Обратная: форма + пример ("Eu fiz o trabalho.")

**Режимы тренировки:**
- Только новые слова (первый раз)
- Только на повторение (по расписанию SM-2)
- Режим ввода — вводишь перевод руками, не выбираешь
- Аудио режим — TTS произносит, вводишь что услышал (диктант)

### 4. ТЕСТЫ

**Режим 1 — Multiple Choice (4 варианта):**
- Правильный ответ + 3 дистрактора из той же категории
- После ответа показывается пример предложения
- Дистракторы подбираются по похожести (comer / falar — плохо; falar / fugir / fechar — хорошо)

**Режим 2 — Ввод перевода:**
- Видишь слово PT → вводишь перевод RU (или наоборот)
- Частичный зачёт: "говорит" вместо "говорить" засчитывается

**Режим 3 — Контекст с пропуском:**
- "Eu ___ português todos os dias."
- Слова берутся из Video Cards (реальный контекст)

**Режим 4 — Конструкции:**
- "Как сказать 'только что сделал'?"
- Ввод: `acabar de + infinitivo`

**Режим 5 — Аудио диктант:**
- TTS произносит слово
- Вводишь что услышал по-португальски
- Тренирует восприятие на слух

### 5. ГРАММАТИКА

**Структура каждой темы = 4 блока:**
1. **Объяснение** — текст + визуальная временна́я шкала (SVG)
2. **Примеры** — реальные фото (Unsplash) + подпись предложением на PT
3. **Таблица спряжений** — интерактивная, каждая ячейка кликабельна, TTS
4. **Упражнения** — fill-in + multiple choice + составь предложение

**Conjugador (спрягатель глаголов):**
- Поле ввода → введи любой глагол
- Автоматически определяет тип: -AR / -ER / -IR / неправильный
- Строит полную таблицу: Presente, Perfeito, Imperfeito, Futuro, Subjuntivo, Imperativo
- Логика правильных глаголов на JS
- Неправильные глаголы: JSON словарь (ser, estar, ter, ir, fazer, poder, querer, vir, ver, dar, saber...)
- Кнопка **▶ Послушать** на каждой форме (Web Speech API TTS)
- Кнопка **+ В карточки** — добавляет колонку как флеш-карточки спряжения

**Визуальные временны́е шкалы (SVG, встроены в HTML):**
```
──────────────────────────────────────────▶ время
   [imperfeito]   [perfeito]        [presente]
   ~~~~~~~~~~~       •                  |
   длительное     точечное           сейчас
   прошлое        прошлое
```

**Мнемо-картинки для неправильных форм:**
- FAZER → fiz: картинка "физически что-то делает" → "ФИЗ как физкультура"
- VIR → vim: стрелка которая приходит к тебе
- Источник картинок: Unsplash API по ключевому слову

**Грамматические карточки из упражнений:**
- Кнопка "+ В карточки" на каждом упражнении
- Добавляет паттерн/конструкцию как отдельную карточку

**Импорт из PDF учебника (продвинутый этап):**
- Загрузка PDF через `pdf.js`
- Парсер вытаскивает: таблицы спряжений → Грамматика, списки слов → Словарь
- Позволяет загрузить Novo Avenida Brasil, Falar Ler Escrever, Panorama Brasil и др.

### 6. ПРОИЗНОШЕНИЕ (ElevenLabs)

**Эталонное TTS произношение:**
- API: ElevenLabs с бразильским голосом
- Каждая карточка имеет кнопку ▶ для прослушивания
- Кэшировать аудио в IndexedDB чтобы не дёргать API повторно

**Запись и проверка произношения:**
```
1. ElevenLabs генерирует эталон слова/предложения
2. Нажимаешь [🎤 Повторить] → MediaRecorder записывает голос
3. Сравнение через ElevenLabs Speech-to-Text или Whisper API
4. Оценка: фонема за фонемой + общий процент совпадения
5. Подсказка: где конкретно ошибка (ударение, гласные, etc.)
```

**Карточка произношения:**
```
"falar"
──────────────────────────
Эталон:  fa-LAR  ✅
Твоё:    FA-lar  ⚠️
──────────────────────────
Ударение: неправильное
Совет: ударение на последний слог в инфинитивах -AR
Оценка: 72% 🟡

[▶ Эталон] [🎤 Повторить] [▶ Послушать себя]
```

**Экспорт в MP3 для зубрёжки:**

Настройки перед экспортом:
- Колода: все слова / только глаголы / на повторение сегодня
- Голос: мужской BR / женский BR (ElevenLabs)
- Пауза между словами: 1с / 2с / 3с
- Направление: PT→RU / RU→PT / оба
- Повторений каждого слова: 1 / 2 / 3

Форматы:
- **Простой список**: `[пауза] falar [пауза] говорить [пауза]`
- **Контекстный**: `[пауза] falar [пауза] "Eu gosto de falar português" [пауза] говорить`
- **Активное вспоминание**: `[пауза] falar ... [долгая пауза] ... говорить`

Техническая реализация:
- ElevenLabs генерирует аудио каждого слова
- Web Audio API склеивает файлы + тишина между ними
- `ffmpeg.wasm` (CDN) → конвертация в MP3 прямо в браузере
- Скачивается как `portuprep-deck-2026-04-10.mp3`

### 7. СТАТИСТИКА И ТРЕКИНГ

**Дашборд "Мой словарный запас":**
```
Estimated vocabulary: ~420 words  ████████░░ B1 level

A1 words known: 98%  ████████████ 245/250
A2 words known: 76%  █████████░░░ 380/500
B1 words known: 34%  ████░░░░░░░░ 340/1000
B2 words known: 12%  █░░░░░░░░░░░ 240/2000
```
- Основа: встроенный JSON частотного списка бразильского PT (top-5000)
- Каждое слово в словаре сопоставляется с уровнем CEFR
- Блок "Следующие 20 слов до B1/B2" — приоритет для изучения

**Heat Map активности (как на GitHub):**
- Календарь за последние 12 месяцев
- Каждый день занятий = зелёная клетка
- Интенсивность = количество повторённых слов
- Хранится в IndexedDB

**График прогресса:**
- Слов добавлено по неделям (bar chart)
- Точность тестов по дням (line chart)
- Серия дней подряд (streak counter)

**Карточка слова с историей:**
```
"falar" — добавлено 14 дней назад
Источник: Português com Letícia EP12
Повторений: 8 раз
Точность: 87%
Следующее повторение: завтра
Интервал: 12 дней
Уровень памяти: ████████░░ Strong
```

**Слабые места (автоматически):**
- Система группирует слова которые постоянно путаешь
- Примеры: "ты часто путаешь ser/estar", "глаголы на -IR даются сложно"
- Кнопка "Тренировать слабые места" → целевая сессия

**Сравнение периодов:**
```
Эта неделя vs прошлая:
Новых слов:      +23  ↑ vs +15
Точность тестов: 78%  ↑ vs 71%
Время занятий:   45мин ↓ vs 62мин
```

**Дашборд "Мой день" (главный экран):**
- X слов на повторение сегодня (SM-2 расписание)
- 1 грамматическая тема дня
- Streak: N дней подряд
